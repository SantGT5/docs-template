# Introduction

Welcome! This is a clean, component-driven documentation template built with
**React**, **Vite** and **Chakra UI**. Clone it, change a few config values, and
start writing your docs in MDX.

<Callout type="tip" title="Why this template?">

You write prose in Markdown, but you can drop **real React components** right into
the page — interactive examples, callouts, copy-to-clipboard commands and more.

</Callout>

## Features

- **MDX authoring** — Markdown plus JSX components, with standard Markdown
  auto-styled to match the theme.
- **Three-column layout** — a fixed header, a navigation sidebar, and an
  automatic "On this page" table of contents.
- **Dark & light mode** — driven by `next-themes`, with a toggle in the header.
- **Reusable content components** — `CodeBox`, `CommandBox`, `Callout` and
  `Steps`, ready to use in any page.
- **Single source of truth** — one `navigation.ts` file powers the sidebar, the
  routes, and prev/next paging.

## How it fits together

| Area           | Where it lives                 |
| -------------- | ------------------------------ |
| Site metadata  | `src/config/site.ts`           |
| Navigation     | `src/config/navigation.ts`     |
| Doc pages      | `src/pages/docs/content/*.mdx` |
| Theme & accent | `src/theme/index.ts`           |
| Components     | `src/components/content/*`     |
